# PubSearcher
PubSearcher é uma aplicação web que permite aos usuários marcar e visualizar bares em um mapa interativo.

### Features
- Login com usuário e senha
  ![image](https://github.com/custodiotiago1986/LitraoSearcher/assets/84323433/fa20c0c3-b8e5-401b-828e-643c73230c8b)

- Inserção de bares no mapa
![image](https://github.com/custodiotiago1986/LitraoSearcher/assets/84323433/33a4e5fe-5aa6-4370-a5c8-f13cef891d0a)

- Visualização de bares inseridos por outros usuários
![image](https://github.com/custodiotiago1986/LitraoSearcher/assets/84323433/bd73d8c0-4ee4-49e6-bbfb-93f0f5b02cc4)

- Permitido exclusão somente de bares inseridos pelo próprio usuário
![image](https://github.com/custodiotiago1986/LitraoSearcher/assets/84323433/324a19c0-3ee0-4509-836c-32b4a07be072)

### Outras características
- Usa localStorage
- HTML5
- CSS3 (Bootstrap 4.5.2)
- JavaScript (jQuery, Leaflet.js com OpenStreetMap, Leaflet Draw, SweetAlert2)
